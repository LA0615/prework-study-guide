HTML

  • The head element contains information about the webpage.
  • The body element represebts the visible content shown to the uswer
  • Metadata cannot be seen by the user
  • The SRC defines the location of an image
  • HTML allows the developer to include text elements, image elements, header elements , and list elements
  • Opening and closing tags are needed to start and end features
  • Declarations are not case sensitive, HTML elements are case sensitive

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • There are 3 types of CSS style sheets: inline, internal, and external
  • Most common way to style a webpage is by using an external CSS style sheet
  • Debugging is very helpful and should be done regularly

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit : commits the command
  • Git Pull Request, PR, is used to merge a comparision branch or feature branch into the base or main branch
  • git commands are NOT the same as BASH commands, cd, or mkdir commands
  • -A indicates that we need to add all changes
  • git checkout "name of branch" will take you to that specific branch

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A string is all the topics that you want in your code.
  • A console.log outputs a message to the web console by adding an argument
  • Chrome DevTools allows you to view the console.og output and helps debug any issues
  • In JavaScript, it is read from top to bottom and ORDER COUNTS
  • Conditional statements and loops help control the order in which code is executed
  • An array is a group of related data in a variable and we use "for" loop to perform a task on each piece of the data using just a few lines of code.
  • JavaScript arrays are zero indexed and sequential.
  • For loop contains a starting point for a loop. We declare a variable, x, and give it a value of 0.
